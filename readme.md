# Weather Web App 🌤

## Project Description

This project is a simple weather web application built using **HTML, CSS, and JavaScript**.
It allows users to search for any city and view the current weather conditions.

The application fetches real-time weather data from the **OpenWeatherMap API** and displays key information such as temperature, weather description, and humidity.

---

## Features

* Search weather by city name
* Displays current temperature in Celsius
* Shows weather condition description
* Displays humidity level
* Weather emoji based on weather conditions
* Responsive and clean user interface
* Press **Enter** or click **Search** to get results

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Fetch API)
* OpenWeatherMap API

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.

No installation or server setup is required.

---

## API Setup

This project uses the **OpenWeatherMap API**.

Steps to get your API key:

1. Create a free account at https://openweathermap.org
2. Navigate to **API Keys** in your account dashboard
3. Copy your API key
4. Replace the placeholder in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

---

## Example Usage

1. Enter a city name (e.g., Delhi, London, Tokyo).
2. Click **Search** or press **Enter**.
3. The application will display the current weather details.
